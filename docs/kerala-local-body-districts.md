# Kerala Local-Body Pilot Districts

Use this to sequence district-by-district collection after the Kerala state-level MLA/MP pilot.

## Current Pilot
- `Ernakulam`

## Suggested Expansion Order
1. `Ernakulam`
2. `Thiruvananthapuram`
3. `Kozhikode`
4. `Malappuram`
5. `Kollam`

## Why This Order
- Starts with districts that have strong urban-rural mix and high public relevance.
- Helps validate:
  - panchayat rows
  - municipality rows
  - corporation rows
  - ward-level source behavior

## Minimum District Deliverable
- one district note file
- one district template CSV
- one verified SEC extraction path
- at least one panchayat member row
- at least one municipal/corporation member row
